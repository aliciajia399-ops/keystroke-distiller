# Keystroke Distiller ⌨️→🧠

> Record every keystroke on your phone, distill your digital self into an AI Skill.
>
> ## What is this?
>
> A mobile-first tool that captures everything you type, stores it locally, and lets you distill your writing patterns into a personalized AI Skill.
>
> ### V1 - Quick Start (Expo React Native)
>
> A simple note-taking app that records all your text input locally.
>
> ```bash
> # 1. Clone
> git clone https://github.com/aliciajia399-ops/keystroke-distiller.git
> cd keystroke-distiller
>
> # 2. Install
> npx expo install
>
> # 3. Run on iOS
> npx expo run:ios
> ```
>
> ### Features
>
> - 📝 Auto-save everything you type
> - - 💾 Local SQLite storage (nothing leaves your phone)
>   - - 🔍 Full-text search across all entries
>     - - 📊 Daily stats (characters, words, sessions)
>       - - 🔌 Local API for Skill distillation
